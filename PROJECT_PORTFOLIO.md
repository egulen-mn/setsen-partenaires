# Tengerly B2B

## À PROPOS

Plateforme B2B de gestion de programmes de parrainage pour restaurants. Transformez vos partenaires commerciaux en ambassadeurs avec un système de suivi automatique, calcul des commissions en temps réel, et tableaux de bord dédiés.

## FONCTIONNALITÉS

→ Génération de QR codes et liens de parrainage personnalisés  
→ Suivi automatique des références via URL tracking  
→ Calcul des commissions en temps réel  
→ Dashboard admin avec gestion complète (partenaires, commandes, paiements)  
→ Dashboard partenaire avec statistiques et outils de partage  
→ Site restaurant démo avec commande en ligne  
→ Interface responsive et multilingue (FR/EN)  
→ Système de paiement des commissions (individuel et en masse)

## TECHNOLOGIES

**Next.js 16** • **React 19** • **TypeScript 5** • **Tailwind CSS 4**

Next.js API Routes • File-based JSON Database • QRCode Generation • PM2 • Nginx • Cloudflare SSL
